# ✨ Gemini Flash API

Aplikasi web AI modern yang menghubungkan **frontend statis** dengan **backend Express.js** dan **Google Gemini API** untuk menghadirkan pengalaman chat multimodal yang cepat dan ringan.

Pengguna dapat berinteraksi dengan AI menggunakan teks maupun berbagai jenis file melalui antarmuka yang sederhana dan responsif.

## 🚀 Fitur

Dengan aplikasi ini, pengguna dapat:

* 💬 Mengirim pertanyaan berbasis teks ke AI
* 🖼️ Mengunggah gambar untuk dianalisis
* 📄 Mengunggah dokumen untuk diringkas atau diproses
* 🎙️ Mengunggah audio untuk ditranskrip menjadi teks
* ⚡ Mendapatkan respons cepat menggunakan model Gemini Flash
* 🌐 Mengakses aplikasi langsung melalui browser tanpa instalasi tambahan

## 📦 Instalasi

Clone repository:

```bash
git clone <repository-url>
cd <project-name>
```

Install dependency:

```bash
npm install
```

Jalankan aplikasi:

```bash
npm run dev
```

Akses melalui:

```plaintext
http://localhost:3000
```
